# [strauh.al](https://strauh.al) is a archive/collection of artworks and images i have downloaded from the internet since 2015/2016. it is the closest thing to my consciousness that i can capture in html.

 the name "strauh.al" comes from my last name and i originally used it as [a link to quickly download python scripts](https://strauh.al/scripts). however, the site grew bigger and eventually there was too much data linked on this website to rename it into something more archival sounding.

 if you're on a mobile device and you enjoy using this site, add it to your homescreen.

 if you're on desktop and you *really* enjoy using this site, you can create a clone of it on your computer by going to [github.com/strauhal/strauh.al3.1](https://github.com/strauhal/strauh.al3.1).

 strauh.al is continually updated and maintained through donations. donations can be made through visiting [strauh.al/donate](https://strauh.al/donate).

 if you have questions or comments about the site, email me at [ernest@strauh.al](mailto:ernest@strauh.al).
