# SEEKER

**A small web tool for browsing the strauh.al image archive — point it at a folder URL (or paste filenames) and it pulls the images.**

## Summary
A utility page within the site that takes a folder URL / plain filenames as input and
displays the contents. It belongs to the same DIY ethos Ernest describes in the diary:
"you don't need to learn to code… just get an llm to figure out the rest for you"
(2025-02-13).
